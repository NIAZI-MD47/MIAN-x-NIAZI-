## 🚀 QUICK START

### Prerequisites
- [Node.js](https://nodejs.org/) v18+
- [Git](https://git-scm.com/)
- [FFmpeg](https://ffmpeg.org/)

### Installation

```bash
# Clone repository
git clone https://github.com/niazi-bot/MIAN-x-NIAZI.git
cd MIAN-x-NIAZI

# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Edit .env with your details

# Start bot
npm start
